## 가장 큰 리스크: 자유 각도 문제

RPD에서 "자유 각도"로 결정했지만, 이것이 PoC에서 가장 큰 도전.

### 문제
- 정면에서 촬영한 그랑 바뜨망 vs 45도 측면에서 촬영한 그랑 바뜨망
- 2D 좌표가 완전히 다름 → L2 Loss가 의미 없는 값을 출력

### 현실적 대응
1. **PoC 단계**: "기준 영상과 비슷한 각도로 찍어주세요" 안내
2. **v0.5 단계**: MediaPipe 3D 좌표 활용 (world_landmarks)
3. **v1.0 단계**: 멀티뷰 또는 깊이 센서 활용

### 결론
자유 각도 완전 대응은 PoC 범위를 벗어남.
PoC에서는 "유사 각도"로 제한하되, 정규화 로직은 미리 설계해둔다.
